Eigen-Stuﬀ

Zhang Ruiyang

[PDF]


We will be discussing all the eigen stuﬀ - eigenvectors, eigenvalues, eigenspace, and etc.

−

A square n×n matrix A can be viewed as a linear transformation of an n dimensional vector space V to itself. A linear transformation is a function, so this square matrix A maps vector x ∈ V to another vector x′∈ V . Some of the vectors of the vector space have their direction unchanged under the linear transformation, and their will at most be a change in scale λ. This, in terms of an equation, is Ax = λx where x is nonzero. We will call such x as eigenvector and its corresponding λ as eigenvalue.

To ﬁnd such eigenvectors computationally wise involve some steps. First, we notice that if we have Ax = λx, we will have (A−λI)x = 0 where this I is the n×n identity matrix. Since x is nonzero, the matrix A−λI must be singular. Singular means not invertible, and the columns of a singular matrix should be linearly dependent in order for it to be not invertible. Now, we recall that a singular matrix will have determinate 0, so detA − λI = 0. Solving this equation should give us the eigenvalues.

−

We should note that the eigenvectors are linearly independent, so they could serve as basis of a vector space. Such vector space is called the eigenspace. In most cases, the number of distinct eigenvectors is the same as the dimension of the vector space, so the dimension of the vector space is the same as that of the eigenspace. Then, we can perform a change-of-basis to the matrix and get something very helpful.

For the n×n matrix A with eigenvalues λ1,λ2,,λn and eigenvectors v1,v2,,vn, we will have the eigenmatrix Q = [v1 v2 vn]. Then, AQ = [λ1v1 λ2v2 λnvn]. Also, if we have Λ the diagonal matrix with the eigenvalues λ1,λ2,,λn on the diagonal, we would have QΛ = [λ1v1 λ2v2 λnvn]. So, we have gotten AQ = QΛ, or A = QΛQ−1. This rewrites the original matrix A using eigen stuﬀ, so it is known as the eigendecomposition of the matrix A.

As mentioned earlier, not all matrices can be decomposed like this. Those that can be eigendecomposed will have the same dimension of eigenspace and vector space, and they are known as diagonalisable, since the process of eigendecomposition will turn into a diagonalisation with a small twist of Q−1AQ = Λ where Λ is a diagonal matrix.
